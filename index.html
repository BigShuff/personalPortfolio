The Technologies & Protocols I Have Used

HTML

Hyper Text Markup Language is one of the fundamental technologies required for web development. It provides the base structure for a web page.

CSS

Cascading Style Sheets define the style and aesthetics of a web page. While HTML is used to structure a web page, CSS specifies the appearance of that structure.

JavaScript

The aforementioned structure is made up of individual elements that users interact with. JS makes it happen.

Git & GitHub

Git is a version control system that facilitates workflow. GitHub on the other hand is an online repository for all previous file versions.

Accessability

A large percentage of internet users can receive a significantly improved experience if the developer follows a few considerations when designing the website.
